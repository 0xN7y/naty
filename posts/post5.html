Linear congruential generator

A linear congruential generator (LCG) is an algorithm that yields a sequence of pseudo-randomized numbers calculated with a discontinuous piecewise linear equation. --wiki.
so first we're gonna use an LCG its like a function You give it a seed, it gives you back numbers that feel random.
Here's the formula:

Xn+1 = (a * Xn + c) % m

Where

  • a is the multiplier
  • c is the increment
  • m is the modulus
  • Xn is the seed (which changes every time)

We will use int value for a,c,m every time we generate and use the above formula

So here’s how it works, straight-up:

1, lcg(seed): This function takes the current seed, runs it through the LCG formula, and spits out a new number. That’s our "random" number generator.

2, randint(low, high, seed): This one gives us a random number between the low and high values you pass in. It updates the seed with each call because you gotta change things up to keep it real.

3, Seed: We start with a seed value (I went with 1234 here, but you do you). Every time you call the randint function, it generates a new "random" number and updates the seed.

This code is raw and it works. Each time you run the loop, it prints a random number between 0 and 100. No libraries, no imports, just Python and math, This is how real coders hustle but Look, This isn't some high-security, unbreakable code. If someone knows your seed and the formula, they can figure out your numbers. So don’t go using this for anything sensitive, like encryption, session key...
